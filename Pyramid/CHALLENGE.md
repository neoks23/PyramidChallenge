# Challenge

This challenge consists of 4 methods that you need to implement in the `PatternGenerator` class. Each method will be tested with various inputs to ensure correctness. The methods are as follows:

The `PatternGenerator` class already contains implementation of `ReverseMarioStairs()`

Reverse Mario Stairs works as follows: Supply integer input of x and return a string that represents a reverse Mario stairs pattern of height x and width r. For example, if x = 4, the output should be:
```
*
**
***
****
```

NOTE: The output should not contain any trailing spaces at the end of each line. Each line should end with a newline character (`\n`), and there should be no extra newline characters at the end of the final output string. Also output syntax in asterisks `*`

The other four methods you need to implement are:

Mario Stairs (x=3):
```
   *
  **
 ***
```

Pyramid (x=3):
```
  *
 ***
*****
```

Steep Pyramid (x=5):
```
    *
   * *
  * * *
 * * * *
* * * * *
```

Diamond (x=3):
```
  *
 * *
* * *
 * *
  *
```
